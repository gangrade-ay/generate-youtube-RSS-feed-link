1) Open any video of the channel and view source code. Search below text via CTRL+F. The ID next to it will be channel ID. Copy the channel ID (do not include inverted commas):

    meta itemprop="channelId" content=
    
    Example:
      The result should look like this:
        meta itemprop="channelId" content="UCbTLwN10NoCU4WDzLf1JMOA"
      Here, UCbTLwN10NoCU4WDzLf1JMOA is the channel ID.

2) Paste it after the below link:

    https://www.youtube.com/feeds/videos.xml?channel_id=
    
    Example:
    The final link should look similar to
        https://www.youtube.com/feeds/videos.xml?channel_id=UCbTLwN10NoCU4WDzLf1JMOA
